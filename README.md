# 360Drone – Tool pratiche ENAC

Repo interno per compilare **ATM-05B** (nulla osta zone P/R/D) e verificare **ATM-09A** (zone aeroporto).

## Workflow rapido (per operatore non pilota)
1. Ricevi ordine → prendi coordinate, data/ora locali, scopo
2. Apri D-Flight → verifica colore zona
   - Celeste 60m / Gialla 45m / Arancione 25m / Rossa 0m → se sopra, serve autorizzazione Specific
3. Se zona P/R → compila Modello ATM-05B (in bollo €16)
4. Invia PEC **almeno 15 gg prima**:
   - SEMPRE in cc: protocollo@pec.enac.gov.it
   - Carcere (P): segreteriasicurezza.dap@giustiziacert.it
   - Città sicurezza (R): protocollo.pref[XX]@pec.interno.it
   - Militare (R): aerosquadra.coa@postacert.difesa.it
